# D204 진진자라


## 주요 기능(MVP)

1. 📱 모바일 앱
    - 콘텐츠 저장
    - 텍스트 : 외부 앱에서 드래그 선택 후 공유하기로 가져옴 , 혹은 사용자가 직접 앱에서 넣을 수 있게도 해야할 듯
    - 캡쳐화면 : 스크린샷 캡쳐 후 공유하기 옵션으로 가져옴 , 이것 또한 앱에서 직접 넣을 수 있도록
        - 스크린샷에서 텍스트가 포함이 될 수 있고 , 포함이 안될 수도 있다. (텍스트가 있다면 유용한 도움이 되는 정보 일 수 있고, 텍스트가 없는 이미지라면, 귀여운 동물 사진이거나 동물 일 수 있다.)
    - 앱 위젯을 만들어서 사용자가 저장 했던 정보나 데이터를 리마인드 해주는 것도 필요하다고 생각한다. ⇒ 정보의 재활용을 위해서 수집한 것이니까 !
2. 💻 크롬 확장 프로그램 
    - 콘텐츠 저장
    - 텍스트 : 웹에서 드래그 선택 후 우클릭, 확장 프로그램 옵션으로 가져옴
    - 이미지 : 웹에서 우클릭
    - 캡쳐화면 : 확장 프로그램 옵션으로 스크린샷 캡쳐로 가져옴
3. 웹 
    - 콘텐츠 저장 및 확인
    - 로컬 pc에 저장된 파일을 업로드 할 수 있도록 하기 위해서 사용자가 직접 파일을 올리는 기능이 있음
4. 🔄 동기화 & 검색
    - 실시간 크로스 플랫폼
        - 모바일에서 저장 → PC에서 즉시 확인 (반대도 가능)
    - 통합 검색 시스템
        - 전체 플랫폼 통합 검색
        - 카테고리, 태그, 날짜
            - 카테고리는 추가, 삭제를 사용자가 커스텀 가능함. (2~10개)
            - 사용자는 회원가입 할 때 원하는 카테고리를 선택과 커스텀이 가능함
            - 태그는 다중 태그로 이미지 혹은 텍스트 연관된 키워드가 태그로 AI를 통해 지정이 됨
    - 스마트 폴더
        - 개발, 진로, 음식 등 자동 분류
            - 태그 기준으로 분류 할 가능성 높아짐
    - 사용자 커스텀 폴더/태그 및 즐겨찾기 기능

### 부가 기능

- 사용자가 이전에 저장했던 정보를 재활용할 수 있도록 도와줘야 한다고 생각한다.
    - 어떻게 다시 볼 수 있게 할 수 있을까 ?
        - **앱에서 있는 위젯 기능 활용**
            - 조회수가 낮은 것들 랜덤 추천 - 즐겨찾기에서 랜덤 추천
            - 안 읽은거 위주로 리마인드 기능
    - 사용자 정보 수집 방식에 따른 타입을 리포트로 알려준다.
- 분류된 데이터를 활용할 수 있는 방안
    - ~~일정 관련 이미지는 캘린더에 등록 될 수 있도록~~
    - ~~주소가 이미지나 텍스트에 있으면 눌렀을때 지도 앱으로 연결 될 수 있도록~~
    - 사용자 기반 챗팅 rag으로 사용자가 저장했던 정보를 기준으로 질문을 하면 저장했던 정보를 찾아준다.
